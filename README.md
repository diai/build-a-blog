# Blog App
## A simple Full-Stack Flask web app for LC-101

# _Implements_:

- [Flask](http://flask.pocoo.org/) template rendering
- manual form validation
- HTTP routing & endpoints
- HTML
- CSS
- request
- redirect
- render_template (in place of jinja2)
- mySQL database
- SQLAlchemy 
- cookie sessions
